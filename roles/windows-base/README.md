windows-setup
=========

Полная конфигурация Windows OS на ПК для рабочего места

Требования
------------

Target-node:
- Установленная Windows OC на таргете.
- Запущенные и сконфигурированые службы WinRM и Remote PowerShell
- Подключение к сети, доступность с Control Node.

Control-node:
- ansible package
- Библиотеки для WinRM (pywinrm)
- Коллекция ansible.windows
